# Text To Speech

This is a simple Text-To-Speech Python program. It Utilizes pyttsx3 and tkinter.

## How to install

```bash
git clone https://github.com/tanishqchejara/text-to-speech.git
cd text-to-speech
pip3 install -r requirements.txt
```

## How to use

```bash
python3 text_to_speech.py
```

## Requirements

- python3>=3.10
- tkinter
- pyttsx3
